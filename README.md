# 🚀 RAG GenAI Project

This is an end-to-end Retrieval-Augmented Generation (RAG) application built using:

- FastAPI
- LangChain
- HuggingFace Embeddings
- FAISS Vector Database
- Streamlit

## Features
✅ Upload documents  
✅ Create embeddings  
✅ Semantic search  
✅ Ask questions from documents  

## Tech Stack
- Python
- FastAPI
- Streamlit
- FAISS
- HuggingFace
- LangChain
